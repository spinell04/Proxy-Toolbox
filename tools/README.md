# Tools

The five modules that make up Proxy Toolbox.

* [IP Uniqueness Test](ip-uniqueness-test.md) — check exit IPs and detect duplicates
* [Ping Test](ping-test.md) — latency tests via TCP, HTTP, or HTTPS
* [TM Request Tester](tm-request-tester.md) — full TLS requests to Ticketmaster
* [Randomize File](randomize-file.md) — shuffle proxy order in a file
* [Proxy Parser](proxy-parser.md) — convert between proxy formats
