# Proxy Toolbox

A CLI toolbox for testing, analyzing, and managing proxy lists — with interactive arrow-key menus and cross-platform binaries.

## What it does

| Tool | Purpose |
|------|---------|
| **IP Uniqueness Test** | Check exit IPs through each proxy and detect duplicates |
| **Ping Test** | Ping a domain through proxies via TCP, HTTP, or HTTPS |
| **TM Request Tester** | Test proxy speed with a full TLS request to Ticketmaster |
| **Randomize File** | Shuffle the proxy order in a file |
| **Proxy Parser** | Convert proxies between different formats |

## Where to start

1. [Installation](getting-started/installation.md) — download the binary and set up folders
2. [Configuration](getting-started/configuration.md) — tweak `config.txt`
3. [Proxy Formats](getting-started/proxy-formats.md) — see what's supported
4. Jump into any tool under **Tools** in the sidebar

## Why this exists

Proxy providers often oversell — selling the same few IPs labelled as thousands of unique endpoints. The toolbox lets you verify what you actually paid for: unique exit IPs, real latency, and how proxies behave against real targets like Ticketmaster.
